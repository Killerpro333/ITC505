Addendum: Lights Out Game Development

Objective:

The Lights Out game challenges players to turn off all the lights on a 5x5 board by clicking tiles. Each click toggles the clicked tile and its adjacent tiles. The game provides a mix of logical strategy and fun, enhanced by its engaging horror-themed design.

HTML Structure:

  1. Game Title: A visually distinct <h1> heading introduces the game with spooky styling.
  2. Game Board: A 5x5 grid of tiles, each styled as a clickable "light," is created dynamically using JavaScript.
  3. Buttons and Stats: Interactive buttons allow starting a new game, and real-time stats for moves and timer are displayed.

CSS Styling:

  1. Background: The addendum page uses a haunting dark theme with glowing accents to match the game's horror feel.
  2. Font Choices: Gothic fonts create an eerie vibe, while complementary colors ensure readability.
  3. Interactive Links: Links have subtle hover effects to guide users while maintaining the horror theme.

JavaScript Logic:

  1. Game Board Generation: JavaScript dynamically generates the board and manages the toggling of tiles.
  2. Move Tracking: The game tracks the player's moves and updates the stats in real-time.
  3. Win/Loss Detection: The game checks for a win (all tiles off) or loss (maximum moves reached) after each move.

Game Enhancements:

  1. Theme Customization: This version uses a horror theme with glowing and dimmed lights for "on" and "off" states.
  2. Responsiveness: The layout adapts to different screen sizes for a seamless experience across devices.
  3. Enhanced Interactivity: Buttons and tiles include smooth hover animations for better feedback.

Conclusion:

The Lights Out game combines simple yet challenging gameplay with a unique horror aesthetic. The project demonstrates the use of dynamic HTML, CSS, and JavaScript to create an engaging user experience.
