Check if a Number is Even or Odd

Lab 6 Project for ITC505

Last updated:

Addendum: This project includes a JavaScript program to check if a number is even or odd. The solution utilizes a simple modulus operation. A challenge encountered was ensuring users entered valid numeric input, which was handled by validating the input using JavaScript's `isNaN` function.
